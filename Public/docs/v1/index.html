Documentation

 

The Joshua Project API is a REST based API. All requests to the API MUST send two pieces of information:

  1. A Valid API Key
  2. A GET Request Method
The server only responds to GET requests! Your URL should be structured as follows:

http://api.joshuaproject.net/v[api_version_number]/[resource_path].[format]?api_key=[your_api_key]&key=value

The current API version is 1. To learn more about the API, check out the Getting Started Tutorial.

Joshua Project is a ministry of the U.S. Center for World Mission. API created by Missional Digerati.
